# Exercicios da Fatec
## Exercicios da matéria de linguagem de programção e algoritimos.
### Todos feitos em C

Nesse repositório existem exercicios com:
* Estrutura de dados
* Usando os tipo de variaveis do C (long, short, double)
* Condicionais
* Ponteiros
* Matrizes (Array)
* Funções
* Maniupulação de Arquivos

## Como rodar
